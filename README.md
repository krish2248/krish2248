## About Me

Machine Learning Engineer and Full-Stack Developer building intelligent systems through code, data, and curiosity. Currently pursuing a Master's degree in Machine Learning, focusing on deep learning, computer vision, and natural language processing.

My work spans predictive modeling, recommendation systems, and full-stack web applications. I believe strong engineering fundamentals are essential for deploying robust ML systems in production.

### What I'm Working On

- Developing ML-powered diagnostic and prediction tools
- Building recommendation engines for personalized experiences
- Creating interactive data visualization applications
- Contributing to open-source ML/AI projects
